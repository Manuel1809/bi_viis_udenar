# Repositorio BI – Trabajo de Grado

Este repositorio contiene los componentes del sistema de Business Intelligence
desarrollado como parte del trabajo de grado.

## Estructura del repositorio

- /docs → documentación del proyecto (requerimientos, diccionarios, manuales)
- /etl → procesos ETL (Kettle / Hop)
- /dw/sql → scripts SQL del Data Warehouse (dimensiones y hechos)
- /dashboards → configuraciones o capturas de tableros
- /qa → pruebas de calidad de datos y validaciones
